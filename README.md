# Appointment Scheduler

Приложение для записи на встречу. Пользователь выбирает дату, свободный временной слот и оставляет контактные данные для подтверждения записи.

## Stack
- React
- Vite
- JavaScript
- CSS
- localStorage

## Run locally
```bash
npm install
npm run dev
```
